# VitaMood

A calm, non-clinical wellness and mental-health companion for Android.

**VitaMood is not a medical device, not a therapist, and not a crisis service.** If you are in immediate danger, contact your local emergency services or use the in-app "Need help now" button, which links to 24/7 human crisis lines.

## About

VitaMood is a free-forever wellness companion designed around four principles:

1. **Calm over engagement** — no streaks, no push spam, no dark patterns
2. **Safety first** — always-visible crisis resources, opt-in AI, humans are the primary answer
3. **Privacy by design** — minimum data, strict security rules, no analytics trackers by default
4. **Free forever** — no subscriptions, no in-app purchases, no premium tier

The full product plan lives in the project's [PLAN.md](https://github.com/herdoy-dev/vitamood/blob/main/PLAN.md).
